Free Website Address Setup

You can publish this website for free using GitHub Pages, Netlify, or Cloudflare Pages. A free address will look like yourname.github.io/lay-bare-it-all or laybareitall.netlify.app.

A true custom domain like laybareitall.com usually is not free because the domain name has to be purchased from a domain registrar. Once you buy one, GitHub Pages and Netlify can connect it.
